Dec 2022 -

Analyzing natural language patterns on Twitter

This project analyzes tweets with a specific hashtag using two approaches: real-time search and data warehouse querying. Real-time search is slower but cheaper, while querying the data warehouse is faster but more expensive. We demonstrated this approach by analyzing 5 hashtags related to the Iranian Revolutionary movement in 2022. We cleaned the tweets' text and created a simple GUI for users to select hashtags. The analysis is basic but can be improved in the future.
