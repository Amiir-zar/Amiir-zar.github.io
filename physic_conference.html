Developing a Web server for the 34th Annual Physics conference

The Imam Khomeini International University in Qazvin,Iran hosted the 34th Annual Physics Conference and the 23rd Physics Student Conference in August 2018.

We initiated this project to improve the experience of conference attendees. We noticed that attendees often faced difficulty navigating through the conference program, finding the location of the sessions, and networking with other attendees. Therefore, we aimed to create a web page that would provide comprehensive information about the conference, including the program schedule, location details, and guidelines for attendees to enhance their experience.

To accomplish this goal, we created a web page that was user-friendly, accessible, and easy to navigate. We ensured that the web page included all relevant information about the conference, such as the program schedule, speaker information, and location details. Additionally, we designed the web page to be mobile-responsive so that attendees could access the information on the go.

To run the web page, we set up a local server that allowed us to test and modify the page before publishing it. We also conducted usability testing to gather feedback and ensure that the web page met the needs of the attendees.

As a result of this project, attendees were able to access all relevant information about the conference in one place. This made it easier for them to plan their schedule, find the location of the sessions, and network with other attendees. The user-friendly design of the web page and the guidelines provided enhanced the overall conference experience for the attendees. Additionally, the use of a local server and usability testing ensured that the web page was error-free and met the needs of the attendees.

In the end, I would like to mention that I have obtained two certifications in Linux as a result of completing this project: the Linux Foundation System Administrator certification and the Linux Implementing the Web Server certification. These certifications serve as evidence of my expertise and experience in Linux and further demonstrate my ability to successfully complete complex projects.
