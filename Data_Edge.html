Data Edge Bussiness Analysis



The goal of this project is to build a comprehensive and detailed Tableau dashboard that provides a holistic view of the company's business performance in the present, historical context, and future projections. The dashboard aims to present key insights to the management, highlighting areas of improvement, and recommending additional data sources for a more comprehensive analysis.

The project involves the following key steps:


Data Cleaning and Preparation:

We took essential steps to ensure the accuracy and consistency of the data. This included removing duplicates, handling missing values, standardizing formats, and structuring the data appropriately. These actions optimized the data for visualization and analysis in Tableau.


Tableau Dashboard Design and Development:

Once the data was cleaned and prepared, we focused on creating a visually appealing and user-friendly Tableau dashboard. We utilized Tableau's powerful visualization capabilities to present a comprehensive overview of our business performance. The dashboard incorporated various charts, graphs, and visual elements that highlighted key metrics and KPIs aligned with Data Edge's objectives. Additionally, we implemented drill-down functionality for more detailed analysis.
